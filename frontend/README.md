# DeltaGrid frontend

React + TypeScript + Vite UI for [DeltaGrid](../README.md).

```bash
npm install
npm run dev      # http://localhost:5173 — proxies /api to :8000
npm test
npm run build
```

Production: set `VITE_API_BASE` to your deployed API URL (including `/api`). See the root README **Hosting** section.
